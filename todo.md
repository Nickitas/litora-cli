# Доработать до научности

Текущая эрозия — это **геометрическая генерация**, а не **геоморфологическая модель**.

1. **Волновая эрозия** (Wave Action)

**Физика:**
- Волны воздействуют на берег с энергией, зависящей от fetch (расстояния открытой воды), глубины, ветра
- Экспозиция берега к волнам определяется углом падения волн

Реализация:**
- Рассчитать **fetch** для каждой точки (расстояние до противоположного берега в разных направлениях)
- Определить **экспозицию** (какие сегменты открыты к морю, какие в бухтах)
- Применить **направленную эрозию** — открытые мысы размываются сильнее

Сноска: `depth` сейчас не из батиметрии, а из геометрического proxy по nearshore openness. Если хочешь, следующим шагом могу добавить реальную батиметрию/грид глубин и заменить этот proxy на физический расчёт.

2. **Осадки и транспорт наносов** (Sediment Transport)

**Физика:**
- Размытый материал не исчезает — он переносится вдоль берега (longshore drift)
- Осадки оседают в зонах затишья (бухты, мысы)
- Баланс эрозии/аккумуляции определяет форму берега

Реализация:**
- Добавить **sediment budget** — сколько материала размыто vs отложено
- Реализовать **longshore drift** — транспорт вдоль берега
- Зоны депозиции → **аккумуляция** (берег растёт, а не только размывается)

3. **Литология** (Lithology / Rock Resistance)

**Физика:**
- Разные породы размываются с разной скоростью:
  - **Известняк/песчаник** — мягкие, быстрая эрозия
  - **Гранит/базальт** — твёрдые, медленная эрозия
  - **Глинистые берега** — очень быстрая эрозия
  - **Галечные пляжи** — динамическое равновесие

Реализация:**
- Создать **lithology profile** для Черного моря (по сегментам)
- Интегрировать в `erosion.go` — сдвиг точки ∝ `1 / rockResistance`
- Визуализировать **разную эрозию** по типу пород (цветовая кодировка в SVG)

4. **Временная динамика** (Temporal Dynamics)

**Физика:**
- Эрозия — процесс во времени (годы/десятилетия)
- Штормовые события vs фоновая эрозия
- Sea-level rise (подъём уровня моря) усиливает эрозию
- Сезонные колебания (зимние штормы vs летний штиль)

Реализация:**
- Добавить **временные шаги** (годы вместо абстрактных "шагов")
- **Штормы** — случайные события с вероятностью и силой
- **Sea-level rise** — затопление низких участков берега
- Визуализация **анимации** (серия SVG → GIF)
