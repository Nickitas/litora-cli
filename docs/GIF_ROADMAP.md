### 🔨 Phase 2: Критические элементы научных графиков

#### 2.1 Масштабная линейка (Scale Bar) 🥇
```bash
--gif-scale-bar [auto|10km|50km|100km]
```
**Зачем:** Обязательный элемент научных графиков
**Реализация:**
- Автоматический расчет подходящего масштаба
- Графическая линейка в углу GIF
- Адаптивная длина (10-50% от ширины)
- Подпись в метрах/километрах

**Приоритет:** КРИТИЧНЫЙ для публикаций

#### 2.2 Цветовая шкала (Color Legend) 🥈
```bash
--gif-color-legend [bottom|right|none]
```
**Зачем:** Понимание цветового кодирования
**Реализация:**
- Вертикальная/горизонтальная палитра цветов
- Подписи значений (0m, 50m, 100m эрозии)
- Разделение на эрозию/аккумуляцию
- Полупрозрачный фон для читаемости

**Приоритет:** КРИТИЧЕСКИЙ для интерпретации

#### 2.3 Географические ориентиры (Geographic Labels) 🥉
```bash
--gif-geo-labels [major|all|custom]
```
**Зачем:** Пространственная ориентация
**Реализация:**
- База координат ключевых точек Черного моря:
  - Города: Одесса, Севастополь, Батуми, Варна, Констанца
  - Мысы: Керченский, Таманский, Крымский
  - Заливы: Каркинитский, Каламитский
- Умные подписи (автопозиционирование)
- Опциональные кастомные метки

**Приоритет:** ВАЖНЫЙ для контекста

## 📈 Medium Priority (полезные улучшения)

### Phase 3: Аналитические улучшения

#### 3.1 Карта стрелок эрозии
```bash
--gif-erosion-arrows
```
- Стрелки перпендикулярно берегу
- Размер = интенсивность эрозии
- Цвет по степени изменения

#### 3.2 График эрозии (Erosion Plot)
```bash
--gif-erosion-plot [right|bottom]
```
- Маленький график накопленной эрозии
- Trend line по шагам
- Как в научных статьях

#### 3.3 Временные метки
```bash
--gif-time-stamp
```
- "Year: X" для временной динамики
- Индикаторы штормов ⛈️
- Климатические события

## 📝 Использование в научных публикациях

### Минимальный набор для статьи:
```bash
./lito model erosion \
  --steps 12 \
  --output-gif figure4.gif \
  --gif-scale-bar auto \
  --gif-color-legend bottom \
  --gif-geo-labels major
```

### Идеальный набор для презентации:
```bash
./lito model erosion \
  --steps 15 \
  --target-years 50 \
  --output-gif presentation.gif \
  --gif-scale-bar 50km \
  --gif-color-legend right \
  --gif-geo-labels all \
  --gif-erosion-arrows \
  --gif-erosion-plot right \
  --gif-width 1920 --gif-height 1080
```
