# Оптимизация размера GIF (GIF Compression) ✅

## Обзор

Реализована базовая оптимизация размера GIF через управление количеством цветов в палитре и уровнем сжатия. Это позволяет балансировать между качеством изображения и размером файла.

## Новые CLI флаги

- `--gif-colors` - количество цветов в палитре (0 = авто 16, диапазон 4-256)
- `--gif-compression` - уровень сжатия (low|medium|high)

## Уровни сжатия

### Low (низкое сжатие - максимальное качество)
```bash
--gif-compression low
```
- Максимальное качество изображения
- Полная палитра цветов
- Минимальные артефакты

### Medium (среднее сжатие - по умолчанию)
```bash
--gif-compression medium
```
- Баланс качества и размера
- Стандартная палитра (16 цветов)
- Рекомендуется для большинства случаев

### High (высокое сжатие - минимальный размер)
```bash
--gif-compression high
```
- Уменьшенная палитра цветов
- Оптимизировано для минимального размера
- Возможны артефакты при малом количестве цветов

## Количество цветов

### Рекомендуемые значения
- **4 цвета**: минимальная палитра, максимальное сжатие
- **8 цветов**: базовая палитра для простых графиков
- **16 цветов**: стандартная научная палитра (авто)
- **32 цвета**: улучшенное качество для детальных графиков
- **64+ цветов**: высокое качество для публикаций

### Зависимость размера от цветов
```bash
# 4 цвета - минимальный размер
--gif-colors 4 --gif-compression high

# 8 цветов - компактное изображение
--gif-colors 8 --gif-compression medium

# 16 цветов - стандарт (по умолчанию)
--gif-colors 0 --gif-compression medium

# 32 цвета - улучшенное качество
--gif-colors 32 --gif-compression low

# 64 цвета - высокое качество
--gif-colors 64 --gif-compression low
```

## Примеры использования

### Для веба (минимальный размер):
```bash
./lito model erosion \
  --steps 6 \
  --output-gif web_compact.gif \
  --gif-width 800 \
  --gif-height 600 \
  --gif-colors 8 \
  --gif-compression high \
  --gif-skip 2
```

### Для презентаций (баланс):
```bash
./lito model erosion \
  --steps 8 \
  --output-gif presentation.gif \
  --gif-width 1600 \
  --gif-height 900 \
  --gif-colors 16 \
  --gif-compression medium
```

### Для научных публикаций (качество):
```bash
./lito model erosion \

  --gif-width 2400 \
  --gif-height 1600 \
  --gif-colors 32 \
  --gif-compression low \
  --gif-geo-labels all \
  --gif-colorlegend-pos right
```


## Технические детали

### Управление палитрой
- При `--gif-compression high` палитра уменьшается в 2 раза
- При `--gif-colors 8` используется минимальная научная палитра
- Автоматический выбор оптимальной палитры для каждого режима

### Ограничения
- Минимум: 4 цвета (для базовой визуализации)
- Максимум: 256 цветов (стандарт GIF)
- Авто: 16 цветов (научная палитра по умолчанию)

### Размеры файлов (тестовые данные)
При разрешении 1200x800, 4 шага эрозии:
- Стандартные настройки (16 цветов, medium): ~50KB
- Высокое разрешение 1920x1080: ~82KB
- Минимальное разрешение 640x480: ~21KB
